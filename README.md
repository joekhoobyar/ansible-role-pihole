# ansible-role-pihole

## Overview

Installs cloudflared, pihole, and/or configures pihole DNS entries

## Supported Systems

- [X] Debian
- [ ] Ubuntu (untested)
- [X] Raspberry Pi OS
- [X] Armbian

### Supported arches

- [X] amd64
- [X] arm64
- [ ] arm (untested)
